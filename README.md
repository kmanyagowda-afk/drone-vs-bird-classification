 Drone vs Bird Classification

1.  Project Overview
This project focuses on classifying aerial objects into Bird  and Drone using deep learning.



2. Objectives
- Classify aerial images
- Detect objects (optional)
- Improve accuracy using CNN models

 
 3.  Dataset
- Train, Validation, Test datasets
- Classes:
  - Bird
  - Drone



4.  Models Used
- CNN
- MobileNetV2
- ResNet50



5.  Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Streamlit



6.  Data Preprocessing
- Image resizing
- Normalization
- Data augmentation



7.  Evaluation Metrics

- F1 Score
